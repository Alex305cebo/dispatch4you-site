### 5. Расширение фона для полного покрытия
- **Изменение:** Фон расширен на 120% с отступами -10%
- **Было:** `width: 100%; height: 100%; top: 0; left: 0;`
- **Стало:** `width: 120%; height: 120%; top: -10%; left: -10%;`
- **Причина:** Убрать видимые края фона на всех разрешениях экрана

### 6. Сдвиг нумерации секторов
- **Изменение:** `left: -80px` → `left: -120px`
- **Причина:** Цифры секторов перекрывали контент, сдвинуты дальше влево

### 7. Исправление кликабельности кнопки в заблокированном квизе
- **Проблема:** Кнопка "🎧 Я закончил слушать этот блок" не нажималась когда квиз заблокирован
- **Решение:** Добавлен CSS для разблокировки кнопки:
  ```css
  .quick-check-block.locked > * {
      pointer-events: none;
  }
  
  .quick-check-block.locked .audio-completion-btn {
      pointer-events: auto;
      opacity: 1;
      cursor: pointer;
  }
  ```
- **Результат:** Кнопка теперь кликабельна даже в заблокированном квизе
