# Dispatch4You - Курсы Диспетчера

Образовательная платформа для обучения диспетчеров грузоперевозок.

## 🚀 Быстрый деплой

Для обновления сайта на https://dispatch4you.com/:

```powershell
.\deploy.ps1
```

Подробная инструкция: [БЫСТРЫЙ-СТАРТ.md](БЫСТРЫЙ-СТАРТ.md)

## 📁 Структура проекта

```
├── index.html              # Главная страница
├── pages/                  # Страницы курсов
│   ├── modules.html        # Список модулей
│   ├── simulator.html      # Симулятор переговоров
│   ├── documentation.html  # Документация
│   └── ...
├── audio/                  # Аудио файлы для симулятора
├── includes/               # Общие компоненты (навигация)
├── quiz/                   # Карточки для обучения
├── config.js               # Конфигурация приложения
├── auth.js                 # Авторизация
└── deploy.ps1              # Скрипт деплоя

```

## 🛠️ Технологии

- HTML5, CSS3, JavaScript (Vanilla)
- Git для версионирования
- Hostinger для хостинга
- Google OAuth для авторизации

## 📝 Разработка

1. Внесите изменения в файлы
2. Запустите `.\deploy.ps1`
3. Проверьте на https://dispatch4you.com/

## 🔐 Безопасность

Файлы в `.gitignore` не попадают в репозиторий:
- Базы данных (*.db)
- Credentials (credentials.json)
- Environment переменные (.env)
- Старые FTP скрипты
