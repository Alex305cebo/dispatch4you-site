# Система аналитики для Dispatch Simulator

## Обзор

Добавлена интеллектуальная система аналитики, которая помогает диспетчерам улучшать свои навыки переговоров в режиме реального времени.

## Основные возможности

### 1. Анализ ответов в реальном времени
После каждого ответа диспетчера система показывает:
- **Оценку качества** (Отлично / Нормально / Слабо)
- **Детальный анализ** выбранного ответа
- **Полезные советы** для улучшения коммуникации
- **Предупреждения** о возможных рисках
- **Следующий шаг** - что ожидать от брокера

### 2. Контекстная помощь
Система анализирует контекст разговора и предоставляет специфичные советы для каждого этапа:

#### Этап 1: MC номер
- Оценка готовности и профессионализма
- Советы по предоставлению информации о компании
- Рекомендации по созданию доверия

#### Этап 2: Оборудование
- Анализ полноты информации об оборудовании
- Советы по подтверждению готовности
- Рекомендации по деталям транспорта

#### Этап 3: Ставка
- Оценка стратегии переговоров
- Советы по обоснованию ставки
- Рекомендации по гибкости в переговорах

### 3. Типы подсказок

#### 💡 Полезные советы (зеленые)
Конкретные рекомендации для улучшения коммуникации

#### ⚠️ Предупреждения (красные)
Важные моменты, которые могут привести к потере груза

#### ℹ️ Информация (синие)
Дополнительная информация о процессе и ожиданиях

### 4. Переключатель аналитики
Пользователи могут включать/выключать систему аналитики:
- Кнопка-переключатель в верхней части чата
- Настройка сохраняется между сессиями
- Позволяет тренироваться без подсказок для опытных пользователей

## Примеры аналитики

### Отличный ответ
```
📊 Оценка: ✓ Отлично
Отличный ответ! Вы предоставили MC номер быстро и добавили важную информацию о компании.

💡 Полезные советы:
✓ Вы показали профессионализм, имея MC номер под рукой
✓ Дополнительная информация о компании создает доверие
✓ Упоминание региона работы показывает опыт на маршруте

🎯 Следующий шаг:
Брокер проверит ваш MC и продолжит с деталями груза. Будьте готовы обсудить ставку.
```

### Слабый ответ
```
📊 Оценка: ✗ Слабо
Слабый ответ. Отсутствие готовности с MC номером создает негативное впечатление.

💡 Полезные советы:
✓ Всегда имейте MC номер под рукой перед звонком
✓ Подготовьте информацию о компании заранее
✓ Неуверенность может привести к потере груза

⚠️ Предупреждение:
Брокер может потерять интерес или перейти к другому перевозчику. Будьте готовы быстро восстановить доверие.
```

## Технические детали

### Хранение данных
- Настройки аналитики сохраняются в `localStorage`
- Ключ: `dispatchSimulatorAnalytics`
- Значение: `true` / `false`

### Структура данных
```javascript
analyticsData = {
    mcNumber: { good, normal, bad },
    equipment: { good, normal, bad },
    rate: { good, normal, bad },
    general: { good, normal, bad }
}
```

### Адаптивность
Система полностью адаптирована для всех размеров экранов:
- Desktop: полная панель с деталями
- Tablet: оптимизированные размеры
- Mobile: компактный вид с сохранением функциональности

## Преимущества

1. **Обучение в реальном времени** - немедленная обратная связь
2. **Контекстная помощь** - советы специфичны для каждой ситуации
3. **Гибкость** - можно отключить для самостоятельной практики
4. **Прогресс** - помогает понять сильные и слабые стороны

## Использование

1. Начните диалог с брокером
2. Выберите ответ из предложенных вариантов
3. Система автоматически покажет анализ вашего ответа
4. Прочитайте советы и примените их в следующих ответах
5. При необходимости отключите аналитику переключателем
