🧪 Navigation Test Page

This page tests if the shared navigation menu is loading correctly.

✅ Checklist

  • ✓ Do you see a navigation bar at the top of the page?
  • ✓ Does it have a logo "🎓 Курсы Диспетчера" on the left?
  • ✓ Are there menu items like "Главная", "Курсы", "Инструменты"?
  • ✓ On desktop: Do dropdown menus appear when you hover over "Курсы"?
  • ✓ On mobile: Do you see a hamburger menu (☰) button?
  • ✓ On mobile: Does clicking the hamburger open a side menu?

🔍 Debug Information

Open Browser Console (F12) to see detailed logs:

  • Look for: === Header Loading Script Started ===
  • Look for: ✓✓✓ Header loaded successfully ✓✓✓
  • Check for any red error messages

📋 Files Required

  • shared-nav.css - Navigation styles
  • shared-header.html - Navigation HTML
  • load-header.js - Loading script

🎯 Next Steps

If the navigation appears correctly here, it should work on all pages that have:

  1. <link rel="stylesheet" href="shared-nav.css"> in the <head>
  2. <div id="header-placeholder"></div> at the start of <body>
  3. <script src="load-header.js"></script> before closing </body>
