# Changelog

## [1.1.0] - Аудио возможности

### Добавлено

#### Новые компоненты
- **AudioButton** - универсальная кнопка для воспроизведения аудио
  - Поддержка 3 размеров (sm, md, lg)
  - Два варианта отображения (inline, block)
  - Прогресс-бар воспроизведения
  - Отображение текущего времени и длительности
  - Автоматическая пауза при завершении

- **TextWithAudio** - компонент-обертка для текста с аудио
  - Автоматическое размещение кнопки Play
  - Опциональный заголовок
  - Поддержка любого React контента

#### Новые страницы
- **/demo** - демонстрация всех аудио возможностей
  - Примеры разных размеров кнопок
  - Варианты отображения
  - Интеграция в текстовый контент
  - Множественные аудио блоки

#### Обновленные страницы
- **Главная страница** - добавлены кнопки "Прослушать превью" на карточки уроков
- **Страница урока** - добавлены:
  - Кнопка "Прослушать весь урок" в заголовке
  - Аудио кнопки для каждого раздела текста
  - Компоненты TextWithAudio для описания и содержания

#### Документация
- **AUDIO-FEATURES.md** - полное описание аудио возможностей
- **QUICK-START.md** - быстрый старт и инструкции
- **CHANGELOG.md** - история изменений

### Улучшено
- Sidebar - добавлена ссылка на демо страницу с иконкой Sparkles
- README.md - обновлен список возможностей и примеры использования
- Навигация - добавлена кнопка "Назад" с иконкой на странице урока

### Технические детали
- Использование React hooks (useState, useRef, useEffect)
- Поддержка HTML5 Audio API
- Адаптивный дизайн для всех компонентов
- TypeScript типизация для всех props
- Tailwind CSS для стилизации
