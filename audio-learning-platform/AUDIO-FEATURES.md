# Аудио Возможности Платформы

## Обзор

Платформа поддерживает воспроизведение аудио на каждой странице и для каждого текстового блока.

## Компоненты

### 1. AudioButton

Универсальная кнопка для воспроизведения аудио.

**Возможности:**
- Воспроизведение/пауза
- Отображение прогресса
- Показ текущего времени и длительности
- Адаптивные размеры
- Два варианта отображения (inline/block)

**Использование:**

```tsx
import AudioButton from '@/src/components/ui/AudioButton';

// Маленькая кнопка
<AudioButton 
  audioUrl="/audio/lesson-1.mp3" 
  label="Прослушать"
  size="sm"
/>

// Средняя кнопка (по умолчанию)
<AudioButton 
  audioUrl="/audio/lesson-1.mp3" 
  label="Прослушать урок"
  size="md"
/>

// Большая кнопка
<AudioButton 
  audioUrl="/audio/lesson-1.mp3" 
  label="Прослушать весь урок"
  size="lg"
  variant="block"
/>
```

### 2. TextWithAudio

Компонент-обертка для текстового контента с кнопкой воспроизведения.

**Возможности:**
- Автоматическое размещение кнопки Play
- Опциональный заголовок
- Поддержка любого React контента

**Использование:**

```tsx
import TextWithAudio from '@/src/components/ui/TextWithAudio';

<TextWithAudio 
  audioUrl="/audio/section-1.mp3"
  title="Введение"
>
  <p>Ваш текстовый контент здесь...</p>
  <ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
  </ul>
</TextWithAudio>
```

## Примеры использования

### На странице урока

```tsx
// Главная кнопка для всего урока
<AudioButton 
  audioUrl={lesson.audioUrl} 
  label="Прослушать весь урок"
  variant="block"
  size="lg"
/>

// Для отдельных разделов
<Card>
  <TextWithAudio 
    audioUrl={lesson.audioUrl}
    title="Описание урока"
  >
    <p>{lesson.description}</p>
  </TextWithAudio>
</Card>
```

### На главной странице

```tsx
// В карточке урока
<Card>
  <h3>{lesson.title}</h3>
  <p>{lesson.description}</p>
  
  <AudioButton 
    audioUrl={lesson.audioUrl} 
    label="Прослушать превью"
    size="sm"
  />
  
  <Button>Начать урок</Button>
</Card>
```

## Структура аудио файлов

Рекомендуемая структура:

```
public/
  audio/
    lesson-1.mp3          # Полный урок
    lesson-1-intro.mp3    # Введение
    lesson-1-section1.mp3 # Раздел 1
    lesson-1-section2.mp3 # Раздел 2
    lesson-2.mp3
    ...
```

## Форматы аудио

Поддерживаемые форматы:
- MP3 (рекомендуется)
- WAV
- OGG
- M4A

## Советы по использованию

1. **Разделяйте длинные уроки** на несколько аудио файлов для удобства навигации
2. **Используйте описательные имена** файлов для легкой идентификации
3. **Оптимизируйте размер** аудио файлов (битрейт 128-192 kbps достаточно для речи)
4. **Добавляйте метаданные** к аудио файлам (название, автор, длительность)
