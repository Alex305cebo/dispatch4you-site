# Эталонная навигация

Этот каталог содержит эталонные файлы навигации, которые используются на всех страницах сайта.

## Файлы

### `navigation.html`
Содержит HTML-разметку навигационного меню:
- Десктопное меню с выпадающими списками
- Мобильное меню
- Кнопки входа и регистрации
- JavaScript для работы мобильного меню

### `navigation-styles.css`
Содержит все CSS-стили для навигации:
- Стили десктопного меню
- Стили мобильного меню
- Анимации и переходы
- Медиа-запросы для адаптивности

## Как использовать

### Вариант 1: Подключение через JavaScript (рекомендуется)

Добавьте в `<head>` вашей страницы:
```html
<link rel="stylesheet" href="../includes/navigation-styles.css">
```

Добавьте в начало `<body>`:
```html
<div id="navigation-placeholder"></div>
<script>
    fetch('../includes/navigation.html')
        .then(response => response.text())
        .then(data => {
            document.getElementById('navigation-placeholder').innerHTML = data;
        });
</script>
```

### Вариант 2: Server-side include (если используется сервер)

PHP:
```php
<?php include '../includes/navigation.html'; ?>
```

Node.js/Express:
```javascript
app.get('/', (req, res) => {
    res.render('page', { navigation: fs.readFileSync('./includes/navigation.html', 'utf8') });
});
```

## Важно!

⚠️ **При изменении навигации редактируйте ТОЛЬКО файлы в папке `includes/`**

Все изменения автоматически применятся на всех страницах, где подключена эта навигация.

## Структура ссылок

Все ссылки в `navigation.html` используют относительные пути от папки `pages/`:
- `../index.html` - главная страница
- `../courses.html` - страница курсов
- `../pages/simulator.html` - страницы в папке pages
- `../login.html` - страницы в корне

Если ваша страница находится в другой папке, скорректируйте пути соответственно.

## Текущие страницы с навигацией

- ✅ `pages/simulator.html` - использует эталонную навигацию

## Добавление навигации на новую страницу

1. Подключите CSS: `<link rel="stylesheet" href="../includes/navigation-styles.css">`
2. Добавьте `padding-top: 60px;` к body (чтобы контент не перекрывался фиксированной шапкой)
3. Подключите HTML навигации через JavaScript или server-side include
4. Проверьте, что все ссылки работают корректно

## Обновление навигации

Чтобы изменить меню на всех страницах:

1. Откройте `includes/navigation.html`
2. Внесите изменения в HTML
3. Сохраните файл
4. Обновите страницы в браузере - изменения применятся автоматически

Для изменения стилей:

1. Откройте `includes/navigation-styles.css`
2. Внесите изменения в CSS
3. Сохраните файл
4. Обновите страницы в браузере
