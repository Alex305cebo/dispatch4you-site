## Содержание модуля:

### 📚 Тема: Типы оборудования и грузов

### Блоки контента (5 аудио-блоков):

1. **🚛 Dry Van - Сухой фургон**
   - Характеристики и размеры
   - Типичные грузы
   - Преимущества и недостатки
   - Средние ставки по маршрутам
   - Таблица с примерами ставок

2. **❄️ Reefer - Рефрижератор**
   - Температурный контроль
   - Типы грузов (продукты, фармацевтика)
   - Дополнительные расходы на топливо
   - Популярные маршруты
   - Средние ставки (выше на 20-40%)

3. **🏗️ Flatbed - Платформа**
   - Открытая платформа для негабаритных грузов
   - Строительные материалы, техника
   - Требования к креплению (tarping, chains)
   - Дополнительные сборы
   - Высокие ставки ($3.00-$3.50/миля)

4. **📐 Step Deck (Drop Deck)**
   - Низкорамная платформа для высоких грузов
   - Два уровня высоты
   - Самые высокие ставки ($3.40-$4.00/миля)
   - Минимальная конкуренция
   - Требует опыта

5. **🚚 Power Only - Только тягач**
   - Работа с чужими прицепами
   - Экономия на покупке прицепа
   - Риски и проверки
   - Популярные клиенты (Amazon, UPS)
   - Хорошие ставки без инвестиций в прицеп

### 📊 Дополнительно:

- **Сравнительная таблица** всех типов оборудования
- Рекомендации для начинающих
- Факторы выбора оборудования
- Средние ставки по всем типам
