## 🎯 Ключевые особенности

### HTML структура:
```html
<nav class="navbar">
  <div class="nav-container">
    <div class="nav-content">
      <!-- Логотип, ссылки, кнопки -->
    </div>
  </div>
</nav>
```

### CSS подключение:
```html
<link rel="stylesheet" href="../shared-nav.css">
<!-- или для корневых страниц -->
<link rel="stylesheet" href="shared-nav.css">
```

### Адаптивность:
- Desktop: полное меню (768px+)
- Tablet: компактное меню (768px-1024px)
- Mobile: гамбургер меню (< 768px)
