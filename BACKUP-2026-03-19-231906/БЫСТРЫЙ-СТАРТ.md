## 📋 Что нужно сделать (5 минут):

### Шаг 1: Настройка Git на Hostinger

1. Войдите в **hPanel** → https://hpanel.hostinger.com/
2. Выберите домен **dispatch4you.com**
3. Найдите раздел **Git** (в Advanced или Files)
4. Нажмите **Create Repository**
5. Заполните:
   - Repository Name: `dispatch4you`
   - Branch: `main`
   - Deploy Path: `/domains/dispatch4you.com/public_html`
   - ✓ Enable Auto Deploy

6. **СКОПИРУЙТЕ** Git Remote URL (выглядит примерно так):
   ```
   ssh://u123456789@123.456.789.0:65002/home/u123456789/repositories/dispatch4you
   ```

### Шаг 2: Подключение к Hostinger

Откройте PowerShell в папке проекта и выполните:

```powershell
# Добавьте remote (замените URL на ваш из Шага 1)
git remote add hostinger ssh://u123456789@123.456.789.0:65002/home/u123456789/repositories/dispatch4you

# Отправьте код на сервер
git push -u hostinger main
```

При первом подключении может запросить пароль от Hostinger.

### Шаг 3: Проверка

Откройте https://dispatch4you.com/ - сайт должен работать!

## ❓ Возможные проблемы

### "Permission denied (publickey)"
Нужно добавить SSH ключ в Hostinger:
1. Сгенерируйте ключ: `ssh-keygen -t rsa -b 4096`
2. Скопируйте: `cat ~/.ssh/id_rsa.pub`
3. Добавьте в hPanel → SSH Keys

### "Repository not found"
Проверьте правильность Git Remote URL в команде `git remote add`

### Сайт не обновляется
1. Проверьте в hPanel → Git → Repository Status
2. Убедитесь что Auto Deploy включен
3. Проверьте Deploy Path: `/domains/dispatch4you.com/public_html`
