## Как это работает

### Шаг 1: Квиз заблокирован
- Квиз серый и неактивный
- Внизу видно сообщение: "🔒 Для активации теста..."
- Кнопка "🎧 Я закончил(а) слушать" видна

### Шаг 2: Пользователь нажимает кнопку
- Квиз разблокируется (становится активным)
- Сообщение и кнопка исчезают
- Квиз прокручивается в центр экрана

### Шаг 3: Пользователь отвечает
- Выбирает вариант ответа
- Получает feedback
- Прогресс сохраняется


```
┌─────────────────────────────────┐
│ ✅ Быстрая проверка             │
│                                 │
│ Вопрос?                         │
│ ● Вариант A                     │ ← активен
│ ○ Вариант B                     │
│ ○ Вариант C                     │
└─────────────────────────────────┘
```
