# ✅ АВТОМАТИЧЕСКАЯ РАЗБЛОКИРОВКА КВИЗОВ

## Что добавлено

Квизы теперь разблокируются автоматически двумя способами:

### 1. При завершении аудио (НОВОЕ!)
Когда аудио-файл доигрывает до конца, квиз автоматически разблокируется

### 2. При нажатии кнопки (уже было)
Пользователь может нажать "🎧 Я закончил(а) слушать" чтобы разблокировать квиз вручную

## Как это работает

### Связь аудио → квиз
```javascript
audio1 → quiz-1  (Блок 1: Роль диспетчера)
audio2 → quiz-2  (Блок 2: Обязанности)
audio3 → quiz-3  (Блок 3: Навыки)
```

### Логика работы

#### Сценарий 1: Аудио доиграло до конца
```
1. Пользователь нажимает ▶ на audio1
2. Аудио начинает играть
3. Прогресс-бар заполняется
4. Аудио заканчивается (onended)
5. ✨ Автоматически вызывается unlockQuizForAudio('audio1')
6. Находится кнопка с data-unlocks="quiz-1"
7. Симулируется клик на кнопку
8. Квиз quiz-1 разблокируется
9. Кнопка меняется на "✓ Завершено"
```

#### Сценарий 2: Пользователь нажал кнопку
```
1. Пользователь нажимает "🎧 Я закончил(а) слушать"
2. Квиз quiz-1 разблокируется
3. Кнопка меняется на "✓ Завершено"
```

#### Сценарий 3: Аудио файл не найден (симуляция)
```
1. Пользователь нажимает ▶ на audio2
2. Файл не найден, запускается симуляция
3. Прогресс-бар заполняется (симулированный)
4. Симуляция заканчивается
5. ✨ Автоматически вызывается unlockQuizForAudio('audio2')
6. Квиз quiz-2 разблокируется
```

## Код

### Функция автоматической разблокировки
```javascript
function unlockQuizForAudio(audioId) {
    // Map audio IDs to quiz IDs
    const audioToQuizMap = {
        'audio1': 'quiz-1',
        'audio2': 'quiz-2',
        'audio3': 'quiz-3'
    };
    
    const quizId = audioToQuizMap[audioId];
    if (quizId) {
        // Find the trigger button for this quiz
        const triggerButton = document.querySelector(`[data-unlocks="${quizId}"]`);
        if (triggerButton && !triggerButton.classList.contains('completed')) {
            // Simulate button click to unlock quiz
            triggerButton.click();
        }
    }
}
```

### Вызов при завершении реального аудио
```javascript
audio.onended = function () {
    // ... cleanup code ...
    
    // Auto-unlock quiz when audio ends
    unlockQuizForAudio(audioId);
};
```

### Вызов при завершении симулированного аудио
```javascript
if (currentTime >= duration) {
    // Audio ended
    // ... cleanup code ...
    
    // Auto-unlock quiz when simulated audio ends
    unlockQuizForAudio(audioId);
}
```

## Преимущества

### 1. Улучшенный UX
- ✅ Пользователь не забывает нажать кнопку
- ✅ Плавный переход от аудио к квизу
- ✅ Автоматизация процесса обучения

### 2. Гибкость
- ✅ Можно слушать аудио до конца (автоматическая разблокировка)
- ✅ Можно пропустить аудио (ручная разблокировка кнопкой)
- ✅ Работает с реальными и симулированными аудио

### 3. Защита от дублирования
```javascript
if (triggerButton && !triggerButton.classList.contains('completed')) {
    // Разблокируем только если еще не разблокировано
}
```
