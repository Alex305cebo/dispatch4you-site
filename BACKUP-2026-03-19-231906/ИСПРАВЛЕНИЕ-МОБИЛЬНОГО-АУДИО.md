# ✅ Исправление конфликта кольца перемотки на мобильных

## 🔴 Проблема
На мобильной версии кольцо перемотки перекрывало кнопку play, из-за чего невозможно было запустить аудио.

## 💡 Причина
- Интерактивное кольцо `.progress-ring-interactive` имело `z-index: 1`
- Кнопка play имела `z-index: 10`
- Но кольцо было интерактивным по всей площади круга, включая центр
- При клике по центру срабатывала функция `seekAudio`, а не `toggleAudio`

## ✅ Решение

### 1. Обновлен CSS для кольца

**Изменения в `.progress-ring-interactive`:**

```css
.progress-ring-interactive {
    z-index: 5;  /* Увеличен с 1 до 5 */
    border-radius: 50%;
}

.progress-ring-interactive.seekable {
    pointer-events: auto;
    /* Создаем кольцо с прозрачным центром для клика */
    background: radial-gradient(
        circle, 
        transparent 0%, 
        transparent 60%, 
        rgba(255,255,255,0.01) 60%, 
        rgba(255,255,255,0.01) 100%
    );
}

/* На мобильных устройствах увеличиваем область клика кольца */
@media (max-width: 768px) {
    .progress-ring-interactive.seekable {
        background: radial-gradient(
            circle, 
            transparent 0%, 
            transparent 55%, 
            rgba(255,255,255,0.01) 55%, 
            rgba(255,255,255,0.01) 100%
        );
    }
}
```

### 2. Обновлен JavaScript для проверки области клика

**Добавлена проверка расстояния от центра:**

```javascript
function seekAudio(event, element, audioId) {
    event.stopPropagation();

    if (!element.classList.contains('seekable')) {
        return;
    }

    const rect = element.getBoundingClientRect();
    const centerX = rect.left + rect.width / 2;
    const centerY = rect.top + rect.height / 2;
    const x = event.clientX - centerX;
    const y = event.clientY - centerY;

    // Calculate distance from center
    const distance = Math.sqrt(x * x + y * y);
    const radius = rect.width / 2;

    // Only allow seeking if click is on the ring (outer 40% of radius)
    // This prevents conflicts with the play button in the center
    if (distance < radius * 0.55) {
        // Click is too close to center, ignore it
        return;
    }

    // ... rest of the function
}
```

## 🎯 Как это работает

1. **Кнопка play** имеет `z-index: 10` (самый высокий)
2. **Кольцо перемотки** имеет `z-index: 5` (средний)
3. **Фоновое кольцо** имеет `z-index: 1` (самый низкий)

4. **CSS градиент** делает центр кольца прозрачным для кликов
5. **JavaScript проверка** игнорирует клики ближе 55% от центра
6. **На мобильных** область прозрачности увеличена до 55% (вместо 60%)

## 💡 Технические детали

### Радиус интерактивной зоны:
- **Desktop**: Центр прозрачен на 60% радиуса
- **Mobile**: Центр прозрачен на 55% радиуса (больше область для перемотки)

### Z-index слои:
```
z-index: 10  → Кнопка play (самый верхний)
z-index: 5   → Интерактивное кольцо
z-index: 1   → Фоновое кольцо SVG
```

### Проверка клика:
```javascript
distance < radius * 0.55  → Игнорировать (слишком близко к центру)
distance >= radius * 0.55 → Разрешить перемотку
```
