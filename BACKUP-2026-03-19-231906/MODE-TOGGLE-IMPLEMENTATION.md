# Реализация переключателя режимов ответа

## Дата: 8 марта 2026 (вечер)

## ✅ ЧТО СДЕЛАНО

Добавлен Toggle Switch для переключения между двумя режимами ответа в модальном окне.

### Визуальный вид:

```
┌─────────────────────────────────────────┐
│ 💡 Выберите ответ                    ✕ │
├─────────────────────────────────────────┤
│                                         │
│ Режим: 🧩 Конструктор ⚪━━━━━ ✅ Выбор  │
│                        ↑                │
│                   (переключатель)       │
│                                         │
│ [Варианты ответов]                      │
│                                         │
└─────────────────────────────────────────┘
```

## 🎯 ДВА РЕЖИМА:

### 1. 🧩 Конструктор (по умолчанию)
- Drag & Drop блоков
- Важен ПОРЯДОК элементов
- Визуальная сборка предложения
- ⭐⭐⭐☆☆ (Легко)

### 2. ✅ Выбор элементов
- Чекбоксы для выбора
- Порядок НЕ важен
- Важно ЧТО включили
- ⭐⭐⭐⭐⭐ (Сложно)

## 💾 СОХРАНЕНИЕ ВЫБОРА

Выбранный режим сохраняется в `localStorage`:
```javascript
localStorage.setItem('answerMode', 'constructor'); // или 'checkbox'
```

При следующем открытии модального окна автоматически загружается сохраненный режим.

## 🎨 СТИЛИ

### Переключатель:
- Фон: `rgba(255, 255, 255, 0.05)`
- Активный режим: `#60a5fa` (синий)
- Неактивный режим: `#94a3b8` (серый)
- Slider: `#334155` → `#60a5fa` при активации

### Анимация:
- Плавное переключение: `transition: 0.3s`
- Slider движется: `translateX(26px)`
- Цвет текста меняется
