## 🔧 Установка на новую страницу

### 1. Добавьте CSS в `<head>`:
```html
<link rel="stylesheet" href="shared-nav.css">
```

### 2. Добавьте placeholder в начало `<body>`:
```html
<div id="header-placeholder"></div>
```

### 3. Добавьте скрипт перед `</body>`:
```html
<script src="load-header.js"></script>
```

### Для страниц в подпапках (например, `/pages/`):
```html
<!-- В <head> -->
<link rel="stylesheet" href="../shared-nav.css">

<!-- Перед </body> -->
<script src="../load-header.js"></script>
```

## ❌ Решение проблем

### Проблема: Меню не появляется

**Диагностика:**
```
1. Откройте консоль (F12)
2. Ищите красные ошибки
3. Проверьте сообщения от load-header.js
```

**Решения:**
- Убедитесь, что все 3 файла существуют (CSS, HTML, JS)
- Проверьте пути к файлам
- Очистите кэш браузера (Ctrl+F5)
- Используйте Chrome или Firefox

### Проблема: Fetch ошибка 404

**Причина:** Файл `shared-header.html` не найден

**Решения:**
- Проверьте, что файл существует в корневой папке
- Для подпапок используйте `../shared-header.html`
- Проверьте регистр букв в имени файла

### Проблема: Стили неправильные

**Решения:**
- Очистите кэш браузера (Ctrl+F5)
- Проверьте, что `shared-nav.css` загружен (вкладка Network)
- Убедитесь, что нет конфликтующих CSS

### Проблема: Мобильное меню не работает

**Решения:**
- Проверьте консоль на JavaScript ошибки
- Убедитесь, что `load-header.js` загружен
- Проверьте, что элементы меню созданы

## 🎨 Кастомизация

### Изменение цветов:
Отредактируйте `shared-nav.css`:
```css
.navbar {
    background: rgba(15, 23, 42, 0.8);  /* Фон навигации */
}

.nav-link {
    color: #94a3b8;  /* Цвет ссылок */
}

.nav-link:hover {
    color: #6366f1;  /* Цвет при наведении */
}
```

### Изменение структуры меню:
Отредактируйте `shared-header.html`:
```html
<div class="nav-dropdown">
    <span class="nav-link">Новый раздел</span>
    <div class="dropdown-content">
        <a href="page1.html">Страница 1</a>
        <a href="page2.html">Страница 2</a>
    </div>
</div>
```
