## 🏗️ СТРУКТУРА КАЖДОГО МОДУЛЯ

### Обязательные элементы:
1. **Header** - навигация с кнопкой "Назад"
2. **Hero Section** - заголовок модуля
3. **Сектора (3-6 на модуль)**:
   - Заголовок сектора
   - Аудио-кнопка с прогрессом
   - Текстовый контент
   - Списки, таблицы, highlight-блоки
4. **Кнопка "Я закончил слушать"** внизу каждого сектора
5. **Навигация** - кнопки "Предыдущий" и "Следующий модуль"
6. **Footer** - стандартный футер

### Дизайн-элементы:
- **Info Highlight** - синие блоки (важная информация)
- **Warning Highlight** - оранжевые блоки (предупреждения)
- **Success Highlight** - зеленые блоки (успехи/советы)
- **Таблицы** - адаптивные таблицы с данными
- **Списки** - маркированные и нумерованные
