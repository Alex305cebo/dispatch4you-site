# 🧪 Локальное тестирование ссылок

## ⚠️ ВАЖНО: Clean URLs не работают локально!

Когда вы открываете файлы через `file://` протокол:
- ❌ `file:///C:/DispatcherTraining/modules` - НЕ РАБОТАЕТ
- ❌ `file:///C:/DispatcherTraining/documentation` - НЕ РАБОТАЕТ
- ✅ `file:///C:/DispatcherTraining/pages/modules-index.html` - РАБОТАЕТ
- ✅ `file:///C:/DispatcherTraining/pages/documentation-new.html` - РАБОТАЕТ

**Почему?** Clean URLs требуют веб-сервер (Apache) с `.htaccess`. Локально нет сервера!

## 📋 Две версии файлов

### 1. index.html (для сервера)
```html
<a href="/modules">Модули</a>
<a href="/documentation">Документация</a>
```
- ✅ Работает на сервере с Apache + .htaccess
- ❌ НЕ работает локально через file://
- 🚀 Загружайте эту версию на хостинг

### 2. index-test-local.html (для локального теста)
```html
<a href="pages/modules-index.html">Модули</a>
<a href="pages/documentation-new.html">Документация</a>
```
- ✅ Работает локально через file://
- ✅ Работает на сервере
- 🧪 Используйте для локального тестирования

## 🔍 Как проверить локально

### Вариант 1: Тестовая версия (РЕКОМЕНДУЕТСЯ)
1. Откройте `index-test-local.html`
2. Кликните на "Модули" или "Документация" в меню
3. Должна открыться нужная страница ✅

### Вариант 2: Прямые ссылки
1. Откройте `БЫСТРЫЙ-ДОСТУП.html`
2. Все ссылки там работают напрямую
3. Выберите нужную страницу ✅

### Вариант 3: Батники
1. Двойной клик на `documentation.bat`
2. Документация откроется сразу ✅

### Вариант 4: Прямой путь к файлу
Откройте напрямую:
- `file:///C:/DispatcherTraining/pages/modules-index.html` - Модули
- `file:///C:/DispatcherTraining/pages/documentation-new.html` - Документация
- `file:///C:/DispatcherTraining/pages/simulator.html` - Симулятор

## 💡 Почему две версии?

**Clean URLs** (`/documentation`) требуют веб-сервер с `.htaccess`:
- На сервере: Apache обрабатывает правила → работает
- Локально: Нет Apache → не работает

**Прямые ссылки** (`pages/documentation-new.html`):
- Работают везде
- Но URL выглядит длиннее
- Не используют преимущества Clean URLs

## ✅ Рекомендация

- Используйте `index-test-local.html`
- Или `БЫСТРЫЙ-ДОСТУП.html`
- Или батники (`documentation.bat`)

### Для загрузки на сайт:

- Загрузите `.htaccess`
