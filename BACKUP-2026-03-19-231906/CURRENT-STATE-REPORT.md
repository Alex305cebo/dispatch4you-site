## 🎯 СТРУКТУРА ДИАЛОГОВ (НОВАЯ)

### 3-Path Structure
1. **master** - хороший путь (11+ шагов)
2. **warning** - вежливое предупреждение (1 шаг с возвратом)
3. **warning_strict** - строгое предупреждение (1 шаг с возвратом)
4. **reject_*_final** - полный отказ (1 шаг с outcome)

### Правила путей
- **Шаги 1-6:** Варианты 3-6 → warning (НЕТ прямого reject!)
- **Шаги 7+:** Варианты 5-6 → reject_final (можно прямой reject)

### Explicit path
- ВСЕГДА указывать `path` в каждом suggestion
- Симулятор использует explicit path, НЕ quality-based fallback
