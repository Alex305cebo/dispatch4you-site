# ⚡ БЫСТРАЯ СПРАВКА - Создание диалогов

## 📋 Структура (14 шагов)

1. MC + Представление
2. Местоположение
3. Детали груза
4. Оборудование
5. Страховка
6. **ТОРГ - Диспетчер называет цену**
7. Встречное предложение
8. Финал + detention
9. Factoring
10. Pickup вопросы
11. Delivery вопросы (опционально)
12. Финальное резюме

## 💎 Шаблон dispatcherPrompt

```
💎 [КОНТЕКСТ]. [ЧТО СКАЗАТЬ: элементы]. [КАК]. [ПОЧЕМУ]!
```

**Примеры:**
- Шаг 1: `💎 Брокер проверяет компанию. Представьтесь: MC, компания, флот, специализация. Чем больше - тем профессиональнее!`
- Шаг 6: `💎 ТОРГ! Posted $X - просите $Y-Z. Обоснуйте: [специфика]. Чем больше - тем больше заработаете!`
- Шаг 8: `💎 ФИНАЛ! $X (просили $Y, posted $Z). Заработали $N! ПОСЛЕДНИЙ шанс - принимайте!`

## 📊 Шаблон analytics

```javascript
"[Emoji] [Оценка]! [Конкретика]. [Детали]."
```

- ✨ Excellent: Все элементы + детали
- ✔️ Good: Основные элементы
- ⚪ Normal: Базовая информация
- ⚠️ Weak: Неполная информация
- 🔴 Aggressive: Грубость
- ❌ Fail: Критические ошибки

## 💰 Формула торга

```
Posted → Request (+15-20%) → Counter → Final (середина)
```

**Пример:**
- Posted: $2,700
- Request: $3,100 (+15%)
- Counter: $2,900
- Final: $2,925 (+$225)

## 🚛 Типы грузов

### Dry Van (50%)
- Rates: $2.30-2.60/mile
- Equipment: Load locks, straps
- Insurance: $100K

### Reefer (35%)
- Rates: $2.75-3.20/mile
- Temp: 34-38°F
- Monitoring: GPS, temp logs

### Hazmat (15%)
- Rates: $3.00-3.50/mile
- Endorsements: Hazmat + Tanker
- Equipment: Placards, emergency kit

### Auto (15%)
- Rates: $2.75-3.00/mile
- Enclosed carrier
- Insurance: $1M+, 360° inspection
