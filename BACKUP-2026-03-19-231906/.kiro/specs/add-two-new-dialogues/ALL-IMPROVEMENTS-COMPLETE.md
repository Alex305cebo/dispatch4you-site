## Ключевые улучшения

### dispatcherPrompt теперь включает:

1. **КОНТЕКСТ** - что происходит на этом шаге
2. **ЧТО СКАЗАТЬ** - конкретные элементы (MC, компания, температура, etc.)
3. **КАК СКАЗАТЬ** - структура ответа
4. **ПОЧЕМУ** - важность (профессионализм, доверие, деньги)
5. **СТРАТЕГИЯ** - тактика для торга (posted → request → final)
6. **МОТИВАЦИЯ** - выгода (заработать $150-600, получить weekly loads)
