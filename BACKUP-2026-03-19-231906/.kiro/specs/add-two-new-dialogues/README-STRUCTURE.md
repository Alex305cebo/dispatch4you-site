## 🎓 Ключевые принципы

1. **dispatcherPrompt** = ГЛАВНАЯ подсказка (КОНТЕКСТ + ЧТО + КАК + ПОЧЕМУ)
2. **analytics** = Обратная связь (конкретика + детали)
3. **Торг** = ЦЕЛЬ диалога (заработать больше)
4. **Специфика груза** = Профессионализм
5. **Weekly loads** = Мотивация

**Каждый диалог должен ОБУЧАТЬ, а не просто тестировать!**
