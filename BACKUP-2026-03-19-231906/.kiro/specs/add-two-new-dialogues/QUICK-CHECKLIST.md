# ✅ БЫСТРЫЙ ЧЕКЛИСТ - ПРОВЕРКА ДИАЛОГА

## 🎯 ПЕРЕД СОЗДАНИЕМ ДИАЛОГА

### Структура
- [ ] 14 шагов (не больше, не меньше)
- [ ] Блок 1: Представление (3 шага)
- [ ] Блок 2: Детали груза (2 шага)
- [ ] Блок 3: Requirements (2 шага)
- [ ] Блок 4: Торг за цену (4 шага) ⭐
- [ ] Блок 5: Финализация (3 шага)

### Качество
- [ ] Каждый шаг = 6 вариантов (Excellent, Good, Normal, Weak, Aggressive, Fail)
- [ ] Брокер реагирует на качество предыдущего ответа
- [ ] Чем хуже качество → тем больше риск отказа

### Реплики
- [ ] Короткие (максимум 2-3 предложения)
- [ ] Профессиональные
- [ ] БЕЗ лишней воды
- [ ] Только ключевая информация

### Торг за цену (КЛЮЧЕВОЕ!)
- [ ] Шаг 8: Брокер спрашивает "What rate?"
- [ ] Шаг 9: Диспетчер называет цену ПЕРВЫМ
- [ ] Чем БОЛЬШЕ просит → тем ЛУЧШЕ качество
- [ ] Шаг 10: Брокер предлагает среднее между posted и запрошенным
- [ ] Шаг 11: Диспетчер финальный ответ

### Requirements
- [ ] Спрашиваются КРАТКО (1-2 вопроса)
- [ ] БЕЗ избыточных деталей
- [ ] Только критичные requirements

### Timing
- [ ] 5-7 минут реального разговора
- [ ] Быстрый темп
- [ ] БЕЗ пауз и длинных объяснений

### Outcome
- [ ] SUCCESS если Excellent/Good/Normal
- [ ] FAILURE если Aggressive/Fail
- [ ] Детальный feedback с уроками

---

## 🚫 ПРОВЕРКА НА ОШИБКИ

### ❌ Длинные реплики?
Если больше 3 предложений → ИСПРАВИТЬ

### ❌ Избыточные детали?
Если спрашивается про каждую деталь → УБРАТЬ

### ❌ Брокер называет цену первым?
Брокер должен спросить "What rate?" → ИСПРАВИТЬ

### ❌ Диспетчер принимает posted rate?
Диспетчер должен просить больше → ИСПРАВИТЬ

### ❌ Нет реакции брокера на качество?
Брокер должен реагировать → ДОБАВИТЬ

### ❌ Timing больше 7 минут?
Сократить реплики → ИСПРАВИТЬ

---

## 💰 ПРОВЕРКА ТОРГА

### Шаг 9 - Диспетчер называет цену:

**Posted rate: $3,200**

- [ ] Excellent: $3,700-3,800 (+$500-600)
- [ ] Good: $3,500-3,600 (+$300-400)
- [ ] Normal: $3,400-3,500 (+$200-300)
- [ ] Weak: $3,250-3,300 (+$50-100)
- [ ] Aggressive: $4,500+ (нереально)
- [ ] Fail: $3,200 (принял posted)

### Шаг 10 - Брокер отвечает:

- [ ] Предлагает среднее: (Posted + Запрошенная) / 2
- [ ] Если Excellent просил $3,800 → Брокер $3,500
- [ ] Если Good просил $3,600 → Брокер $3,400
- [ ] Если Normal просил $3,500 → Брокер $3,350

### Шаг 11 - Диспетчер финальный ответ:

- [ ] Excellent: Торгуется еще
- [ ] Good: Принимает с условием
- [ ] Normal: Принимает
- [ ] Weak: Неуверенно принимает
- [ ] Aggressive: Ультиматум
- [ ] Fail: Отказывается
