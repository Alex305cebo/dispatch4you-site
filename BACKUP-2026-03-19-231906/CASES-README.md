# База данных практических кейсов

## Описание
Динамическая система отображения практических кейсов для диспетчеров грузоперевозок.

## Файлы
- `cases-database.js` - База данных с 12 кейсами
- `index.html` - Главная страница с секцией кейсов

## Функционал
✅ 12 уникальных кейсов в базе данных
✅ Случайная загрузка 4 кейсов при открытии страницы
✅ Кнопка "Показать другие кейсы" для обновления
✅ Современный дизайн карточек с градиентами
✅ Категории: Логистика, Переговоры, Кризис, Документация, и др.
✅ Уровни сложности: Базовый, Средний, Сложный
✅ Теги для быстрой идентификации
✅ Иконки для визуального различия
✅ Решения прямо на карточке
✅ Адаптивный дизайн для мобильных

## Структура кейса
```javascript
{
  id: 1,
  title: "Название кейса",
  description: "Описание ситуации",
  difficulty: "easy|medium|hard",
  category: "Категория",
  icon: "🔧",
  solution: "Решение проблемы",
  tags: ["Тег1", "Тег2"]
}
```

## Как добавить новый кейс
1. Открыть `cases-database.js`
2. Добавить новый объект в массив `casesDatabase`
3. Указать уникальный `id`
4. Заполнить все поля
5. Сохранить файл
