# Инструкция по улучшению анимации слот-машины

## Проблема
Текущая анимация "Случайный диалог" останавливает все 3 слота одновременно - нереалистично.

## Решение
Каждый слот должен останавливаться в разное время с постепенным замедлением, как в настоящем казино.

## Что изменить

В файле `pages/simulator.html` найдите функцию `setupRandomScenarioButton()` (строка ~3307)

Замените весь блок анимации (от `// Анимация слотов` до конца `setInterval`) на код из файла `pages/slot-machine-improved.js`

## Ключевые улучшения:

1. **Разное время остановки**: 
   - Слот 1: останавливается через 1.5 сек
   - Слот 2: останавливается через 2.2 сек  
   - Слот 3: останавливается через 2.9 сек

2. **Постепенное замедление**:
   - Быстрое вращение (80ms между спинами)
   - Замедление за 5 спинов до остановки
   - Каждый следующий спин медленнее: 150ms, 200ms, 250ms, 300ms, 350ms

3. **Bounce эффект при остановке**:
   - `transform: scale(1.1)` → `scale(1)` за 200ms
   - Зеленая подсветка при остановке

4. **Финальный сценарий выбирается заранее**:
   - Все 3 слота показывают ОДИН и тот же финальный сценарий
   - Более реалистично, чем случайные разные сценарии

## Результат
Анимация выглядит как настоящая слот-машина в казино!
