📋 Инструкции для запуска сервера:

Вариант A: PowerShell скрипт

  1. Откройте PowerShell в папке проекта
  2. Запустите: .\start-server.ps1
  3. Откройте: http://localhost:8000/test-nav.html

Вариант B: Python

  1. Откройте командную строку в папке проекта
  2. Запустите: python -m http.server 8000
  3. Откройте: http://localhost:8000/test-nav.html

Вариант C: VS Code Live Server

  1. Установите расширение "Live Server" в VS Code
  2. Откройте test-nav.html
  3. Нажмите правой кнопкой → "Open with Live Server"
