## 🎯 Что нужно настроить

### 1. Включить профиль пользователя

Добавь в любой файл после загрузки навигации:

```javascript
document.addEventListener('navLoaded', () => {
    const profile = document.getElementById('navUserProfile');
    if (profile) {
        profile.style.display = 'flex';
        
        // Обновить данные пользователя
        profile.querySelector('.user-name').textContent = 'Иван Иванов';
        profile.querySelector('.user-progress').textContent = 'Прогресс: 75%';
        profile.querySelector('.notification-badge').textContent = '5';
    }
});
```

### 2. Настроить поиск

Открой `nav-enhanced.js` и найди функцию `performSearch`:

```javascript
function performSearch(query) {
    // Замени на свою логику
    window.location.href = `/search?q=${encodeURIComponent(query)}`;
}
```

## 📱 Как использовать свайп-жесты

### На мобильном устройстве:

1. **Открыть меню:**
   - Проведи пальцем от правого края экрана влево
   - Или нажми на кнопку ☰

2. **Закрыть меню:**
   - Проведи пальцем вправо в открытом меню
   - Или нажми на затемненную область
   - Или нажми кнопку ✕

3. **Haptic feedback:**
   - Телефон вибрирует при открытии/закрытии

---

## ⌨️ Горячие клавиши

- **Ctrl+K** (или Cmd+K на Mac) - Открыть поиск
- **ESC** - Закрыть мобильное меню

## 🐛 Если что-то не работает

### Свайп-жесты не работают:
- Проверь, что открываешь на touch-устройстве
- Убедись, что `nav-enhanced.js` загружен
- Проверь консоль браузера на ошибки

### Поиск не работает:
- Настрой функцию `performSearch()` в `nav-enhanced.js`
- Проверь, что поле поиска видно (desktop)

### Профиль не отображается:
- Включи его через код выше
- По умолчанию профиль скрыт
