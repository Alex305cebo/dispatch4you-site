## Для других модулей

При добавлении интерактивных блоков в другие модули используйте:

### Заголовки:
```html
<h3 class="case-study-title">Реальный кейс: [название]</h3>
<h3 class="quick-check-title">Быстрая проверка</h3>
```

### Кнопки:
```html
<button class="audio-completion-trigger" data-unlocks="quiz-X">
    🎧 Я закончил(а) слушать
</button>
```

### Feedback:
```html
<strong>Правильно! ✓</strong>
```
