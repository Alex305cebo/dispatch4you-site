3. https://dispatch4you.com/doc-module-1
   → Должен открыть pages/doc-module-1.html

4. https://dispatch4you.com/simulator
   → Должен открыть pages/simulator.html

   → Должен открыть страницу быстрого доступа

═══════════════════════════════════════════════════════════════════
  🔧 ВАЖНЫЕ ЗАМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════

1. .htaccess работает только на Apache серверах
   → Если у вас Nginx, нужна другая конфигурация

2. Батники (.bat) не нужны на сервере
   → Они работают только на Windows локально

3. Все HTML страницы должны быть в UTF-8
   → Проверьте кодировку перед загрузкой

4. Права доступа к .htaccess должны быть 644
   → chmod 644 .htaccess
