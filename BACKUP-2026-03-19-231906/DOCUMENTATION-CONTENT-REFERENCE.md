## ⏰ РАБОЧИЙ ДЕНЬ ДИСПЕТЧЕРА

### Утро (06:00-09:00):
- Проверка статуса всех активных грузов
- Связь с водителями для подтверждения местоположения
- Поиск новых грузов на Load Boards
- Проверка времени доставки и возможных задержек
- Планирование маршрутов на день

### День (09:00-17:00):
- Ведение переговоров с брокерами о ставках
- Проверка MC DOT брокеров
- Бронирование грузов для водителей
- Отправка rate confirmations и BOL
- Мониторинг движения траков через GPS
- Решение проблем: поломки, пробки, изменения
- Координация времени погрузки и разгрузки

### Вечер (17:00-20:00):
- Подтверждение доставок за день
- Планирование маршрутов на следующий день
- Поиск грузов для обратного пути
- Обработка документации и инвойсов
- Подготовка отчетов

## 📞 ТЕХНИКИ ПЕРЕГОВОРОВ (из выпадающих блоков)

### 5 продвинутых техник:

1. **Anchoring (Якорение)** - первым называйте цену выше желаемой
2. **BATNA** - всегда имейте альтернативу
3. **Value Stacking** - обоснуйте высокую цену дополнительной ценностью
4. **Silence is Golden** - после цены молчите 5-10 секунд
5. **Package Deal** - предложите несколько грузов за лучшую ставку

### Дополнительные тактики:
- **Mirroring** - повторяйте последние 1-3 слова собеседника
- **Labeling Emotions** - называйте эмоции собеседника
- **Calibrated Questions** - вопросы с "How" или "What"
- **Accusation Audit** - предвосхищайте негативные мысли

## 📋 ОБЯЗАТЕЛЬНЫЕ ДОКУМЕНТЫ

### Для каждого груза:
1. **Rate Confirmation** - перед грузом (от брокера)
2. **BOL (Bill of Lading)** - при погрузке (от грузоотправителя)
3. **POD (Proof of Delivery)** - при разгрузке (от получателя)
4. **Invoice** - после доставки (вы создаете)

### Для компании:
- MC Authority (активный)
- DOT Number (активный)
- Insurance: Liability $750K-$1M, Cargo $100K+
- UCR (ежегодно)
- IFTA (для нескольких штатов)
- BOC-3 (process agent)

## 🔍 ПРОВЕРКА БРОКЕРОВ (5-минутный чек-лист)

### Шаг 1: FMCSA Verification (2 мин)
- Status: Active
- Operating Authority: Broker
- Insurance: $75,000 BMC-84/85 bond
- Дата регистрации: минимум 6+ месяцев

### Шаг 2: Credit Check (1 мин)
- RTS Pro или Carrier411
- Score выше C
- Days to pay <45

### Шаг 3: Online Reputation (2 мин)
- Google reviews
- BBB.org
- TruckersReport.com
- Reddit r/Truckers

### ❌ Признаки мошенников:
- MC# не проходит проверку
- Требуют предоплату
- Ставка на 50%+ выше рынка
- Нет физического адреса
- Email с бесплатных доменов (gmail, yahoo)
- Спешат и давят

## 💪 WORK-LIFE BALANCE

### Признаки burnout:
- Постоянная усталость
- Раздражительность
- Снижение производительности
- Потеря интереса к работе
- Физические симптомы

### Профилактика:
- **Set Boundaries**: четкие рабочие часы (7:00-18:00)
- **Delegate**: не делайте все сами
- **Take Breaks**: каждые 2 часа по 10-15 мин
- **Physical Health**: 30 мин exercise, 7-8 часов сна
- **Mental Health**: медитация 10 мин/день, хобби, терапия

## 🚨 ЭКСТРЕННЫЕ СИТУАЦИИ - ПРОТОКОЛЫ

### Cargo Theft (кража груза):
1. Звоните 911 (получите case number)
2. Уведомите брокера немедленно
3. Звоните в страховую
4. Freeze location (GPS tracking)
5. FBI для грузов >$100K

### Серьезная авария:
1. Убедитесь водитель в безопасности
2. Звоните в страховую НЕМЕДЛЕННО
3. Уведомите брокера
4. НЕ признавайте вину
5. Документируйте все
6. Юрист для серьезных случаев

### Водитель пропал:
- 0-2 часа: звонки, SMS
- 2-4 часа: проверьте GPS, звоните в truck stops
- 4-6 часов: звоните в больницы
- 6-12 часов: missing person report
- 12+ часов: уведомите семью
