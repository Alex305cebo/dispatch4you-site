## Как это работает

### Механика разблокировки:
1. Студент читает/слушает контент
2. Нажимает кнопку "🎧 I've finished listening"
3. Квиз разблокируется (убирается класс `locked`)
4. Студент отвечает на вопрос
5. Получает мгновенную обратную связь (зеленый/красный)
6. Прогресс сохраняется в localStorage

### Технические детали:
- ✅ CSS подключен: `<link rel="stylesheet" href="../interactive-modules.css">`
- ✅ JS подключен: `<script src="../interactive-modules.js"></script>`
- ✅ Атрибут body: `data-module-id="module-1"`
- ✅ Уникальные ID для квизов: `quiz-1`, `quiz-2`, `quiz-3`
- ✅ Правильные data-атрибуты: `data-quiz-id`, `data-correct-answer`, `data-unlocks`
