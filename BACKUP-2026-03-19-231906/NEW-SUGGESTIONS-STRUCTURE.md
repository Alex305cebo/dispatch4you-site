# НОВАЯ СТРУКТУРА КАРТОЧЕК ВАРИАНТОВ ОТВЕТОВ

## ИЗМЕНЕНИЯ

Переработана структура модального окна с вариантами ответов. Теперь карточки имеют более наглядный дизайн с разделением на секции.

## НОВАЯ СТРУКТУРА КАРТОЧКИ

### ДО ВЫБОРА (нейтральный вид):
```
┌─────────────────────────────────────┐
│ [скрытая иконка]    [скрытый бейдж] │ ← Header (цветной фон)
├─────────────────────────────────────┤
│ Текст ответа диспетчера...          │ ← Body
│                                     │
└─────────────────────────────────────┘
```

### ПОСЛЕ ВЫБОРА (показывает качество):
```
┌─────────────────────────────────────┐
│ ✨              [✨ Отлично]         │ ← Header (зеленый фон)
├─────────────────────────────────────┤
│ Текст ответа диспетчера...          │ ← Body
├─────────────────────────────────────┤
│ ✨ ОТЛИЧНО! MC, компания, опыт...   │ ← Analytics (темный фон)
└─────────────────────────────────────┘
```

## ЭЛЕМЕНТЫ КАРТОЧКИ

### 1. Header (верхняя часть)
- **Фон**: Цветной, соответствует качеству ответа
- **Иконка слева**: Скрыта до выбора (opacity: 0)
- **Бейдж справа**: Скрыт до выбора (display: none)
- **Цвета по качеству**:
  - ✨ Отлично: зеленый (#22c55e)
  - ✔️ Хорошо: лайм (#84cc16)
  - ⚪ Нормально: желтый (#eab308)
  - ⚠️ Слабо: оранжевый (#f97316)
  - 🔴 Агрессивно: красный (#ef4444)
  - ❌ Провал: темно-красный (#dc2626)

### 2. Body (текст ответа)
- Полный текст ответа диспетчера
- Белый цвет (#e2e8f0)
- Размер: 15px
- Padding: 16px

### 3. Analytics (аналитика)
- Скрыта до выбора
- Показывается после клика
- Темный фон (rgba(15, 23, 42, 0.5))
- Серый текст (#94a3b8)
- Курсив, 13px

## ПОВЕДЕНИЕ

### До выбора:
- Все карточки выглядят одинаково (нейтрально)
- Серая граница
- Hover: синяя подсветка

### После выбора:
1. ВСЕ карточки показывают свое качество
2. Границы окрашиваются в цвет качества
3. Иконки становятся видимыми
4. Бейджи появляются
5. Аналитика раскрывается
6. Выбранная карточка подсвечивается (glow effect)
7. Через 3 секунды модальное окно закрывается
