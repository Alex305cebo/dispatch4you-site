### Интерактивные элементы:

#### ✅ Аудио-кнопки (7 штук)
- Красивая кнопка с градиентом
- Прогресс-кольцо (desktop)
- Мобильная прогресс-полоса
- Таймер воспроизведения
- Перемотка кликом (desktop)
- Анимация пульсации при воспроизведении

#### 💼 Case Study блоки (7 штук)
- Реальные кейсы из индустрии
- Ситуация + вопрос + решение
- Красивое оформление с иконками
- Hover эффекты

#### ✅ Quick Check квизы (7 штук)
- Блокировка до прослушивания аудио
- 3 варианта ответа (A, B, C)
- Подсветка правильного/неправильного ответа
- Feedback с объяснением
- Красивая анимация разблокировки

#### 🎧 Кнопки "Я закончил слушать" (7 штук)
- Разблокируют квизы
- Меняют состояние на "✅ Блок прослушан"
- Disabled после клика

### JavaScript функции:
- `toggleAudio()` - воспроизведение/пауза
- `updateProgress()` - обновление прогресса
- `seekAudio()` - перемотка на desktop
- `seekAudioMobile()` - перемотка на mobile
- `markAudioComplete()` - разблокировка квиза
- `formatTime()` - форматирование времени
- Quiz обработчики
