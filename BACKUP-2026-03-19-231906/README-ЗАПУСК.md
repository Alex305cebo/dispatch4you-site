# 🚀 Быстрый старт

## ⚠️ Проблема: "Не удалось получить доступ к файлу"

Вы видите эту ошибку, потому что открываете сайт через двойной клик на HTML файл (`file://` протокол). 

**Clean URLs** (например `/modules`, `/simulator`) работают только через веб-сервер!

---

## ✅ Решения (выберите одно)

### 🎯 Вариант 1: Быстрый тест (2 секунды)

**Двойной клик на файл:**
```
ОТКРЫТЬ-ТЕСТ.bat
```

Или откройте вручную: `test-local-navigation.html`

✅ Все ссылки работают сразу!

---

### 🚀 Вариант 2: Запуск сервера (рекомендуется)

**Двойной клик на файл:**
```
ЗАПУСТИТЬ-СЕРВЕР.bat
```

Или в PowerShell:
```powershell
.\start-local-server.ps1
```

Затем откройте: http://localhost:8000

✅ Clean URLs будут работать частично

---

### 💎 Вариант 3: XAMPP (100% работа)

1. Скачайте: https://www.apachefriends.org/
2. Установите и запустите Apache
3. Скопируйте проект в `C:\xampp\htdocs\dispatch4you\`
4. Откройте: http://localhost/dispatch4you/

✅ Clean URLs работают полностью!

---
