# 📚 Эталонная навигация - Полный индекс

## 🎯 Назначение

Централизованная система навигации для всего сайта "Курсы Диспетчера". Изменения в одном месте автоматически применяются на всех страницах.

---

## 📁 Структура файлов

### Основные файлы

| Файл | Назначение | Размер |
|------|-----------|--------|
| `navigation.html` | HTML-разметка навигации | ~4 KB |
| `navigation-styles.css` | CSS-стили навигации | ~6 KB |
| `load-navigation.js` | Автозагрузка навигации (опционально) | ~1 KB |

### Документация

| Файл | Описание |
|------|----------|
| `README.md` | Полная документация с примерами |
| `QUICK-START.md` | Быстрый старт за 3 шага |
| `EXAMPLE.html` | Рабочий пример страницы |
| `CHANGELOG.md` | История изменений |
| `PAGES-TO-UPDATE.md` | Список страниц для обновления |
| `INDEX.md` | Этот файл - общий обзор |

---

## 🚀 Быстрый старт

```html
<!-- 1. В <head> -->
<link rel="stylesheet" href="../includes/navigation-styles.css">

<!-- 2. В начале <body> -->
<div id="navigation-placeholder"></div>

<!-- 3. Перед </body> -->
<script>
    fetch('../includes/navigation.html')
        .then(response => response.text())
        .then(data => {
            document.getElementById('navigation-placeholder').innerHTML = data;
        });
</script>
```

## 🎨 Дизайн

### Цветовая схема
- Фон: `rgba(10, 14, 26, 0.9)` с backdrop-blur
- Текст: `#ffffff` (основной), `#94a3b8` (вторичный)
- Акцент: `#6366f1` (индиго)
- Hover: `rgba(99, 102, 241, 0.15)`

### Анимации
- Выпадающие меню: `fadeInDown` (0.3s)
- Подчеркивание ссылок: плавное появление (0.3s)
- Мобильное меню: slide-in справа (0.3s)

### Адаптивность
- Десктоп: > 768px - горизонтальное меню
- Мобильный: ≤ 768px - боковое меню
