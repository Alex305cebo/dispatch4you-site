# История изменений навигации

Все значимые изменения эталонной навигации документируются в этом файле.

## [1.1.0] - 2026-02-28

### 🔄 Изменено
- Выпадающие меню теперь открываются по клику, а не только по наведению
- Меню остается открытым до выбора пункта или клика вне меню
- Улучшена UX: меню не закрывается случайно при движении мыши

### 🐛 Исправлено
- Исправлена проблема с закрытием выпадающего меню при случайном движении мыши

### 📄 Обновленные страницы
- `includes/navigation.html` - добавлен JavaScript для управления кликами
- `includes/navigation-styles.css` - добавлен стиль для активного состояния
- `pages/simulator.html` - обновлены стили и скрипты

## [1.0.0] - 2026-02-28

### ✨ Добавлено
- Создана эталонная навигация в `navigation.html`
- Создан файл стилей `navigation-styles.css`
- Добавлен скрипт автозагрузки `load-navigation.js`
- Создана документация:
  - `README.md` - полная документация
  - `QUICK-START.md` - быстрый старт
  - `EXAMPLE.html` - пример использования
  - `PAGES-TO-UPDATE.md` - список страниц для обновления
  - `CHANGELOG.md` - этот файл

### 🎯 Структура меню
- Главная
- Курсы (выпадающий список):
  - 12 Модулей обучения
  - Симулятор переговоров
  - Примеры звонков
  - Практические кейсы
- Инструменты (выпадающий список):
  - Поиск Грузов
  - Dispatcher Cards
  - AI Чат
  - Load Board
- Документация
- О нас
- Кнопки: Войти, Регистрация

### 📱 Функциональность
- Адаптивный дизайн (десктоп + мобильный)
- Выпадающие меню на десктопе
- Боковое меню на мобильных устройствах
- Плавные анимации и переходы
- Backdrop blur эффект

### 🎨 Стили
- Темная тема (#0a0e1a)
- Полупрозрачный фон с размытием
- Градиентные эффекты при наведении
- Анимированные подчеркивания ссылок

### 📄 Страницы с навигацией
- `pages/simulator.html` - первая страница с эталонной навигацией
