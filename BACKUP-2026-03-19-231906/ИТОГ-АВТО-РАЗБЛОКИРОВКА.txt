ТЕПЕРЬ КВИЗЫ РАЗБЛОКИРУЮТСЯ ДВУМЯ СПОСОБАМИ:

1️⃣ АВТОМАТИЧЕСКИ (НОВОЕ!)
   ✓ Когда аудио доигрывает до конца
   ✓ Работает для реальных аудио файлов
   ✓ Работает для симулированного аудио
   ✓ Кнопка автоматически меняется на "✓ Завершено"

2️⃣ ВРУЧНУЮ (КАК БЫЛО)
   ✓ Кнопка "🎧 Я закончил(а) слушать"
   ✓ Для тех кто хочет пропустить аудио
   ✓ Для тех кто читает без звука

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

СВЯЗЬ АУДИО → КВИЗ:

audio1 → quiz-1  (Блок 1: Роль диспетчера)
audio2 → quiz-2  (Блок 2: Обязанности)
audio3 → quiz-3  (Блок 3: Навыки)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

КАК ЭТО РАБОТАЕТ:

1. Пользователь нажимает ▶ на аудио
2. Аудио играет, прогресс-бар заполняется
3. Аудио заканчивается
4. ✨ Автоматически разблокируется квиз
5. Кнопка меняется на "✓ Завершено"
6. Пользователь может отвечать на вопросы
