## 🔄 Процесс загрузки

```
1. Страница загружается
   │
   ▼
2. nav-loader.js запускается
   │
   ├─→ Определяет BASE путь (корень или pages/)
   │
   ├─→ Загружает nav.html через fetch
   │
   ├─→ Заменяет {{BASE}} на правильный путь
   │
   ├─→ Вставляет HTML в #nav-placeholder
   │
   ├─→ Инициализирует мобильное меню
   │
   ├─→ Подсвечивает активную страницу
   │
   └─→ Загружает nav-enhanced.js
       │
       ▼
3. nav-enhanced.js инициализируется
   │
   ├─→ Свайп-жесты
   ├─→ Прогресс скролла
   ├─→ Поиск
   ├─→ Горячие клавиши
   └─→ Анимации
```

│  ↓                                  │
│  ↓                                  │
│  ↓                                  │

## 🔧 Зависимости и связи

```
┌─────────────────────────────────────────────┐
│  ФАЙЛЫ И ИХ ЗАВИСИМОСТИ                     │
└─────────────────────────────────────────────┘

nav-loader.js
├─→ Загружает nav.html
├─→ Загружает nav-enhanced.js
└─→ Использует shared-nav.css

nav-enhanced.js
├─→ Требует nav.html (DOM элементы)
├─→ Использует shared-nav.css (классы)
└─→ Работает с nav-loader.js (события)

shared-nav.css
├─→ Независимый файл
└─→ Используется всеми компонентами

nav.html
├─→ Шаблон с {{BASE}} плейсхолдерами
└─→ Обрабатывается nav-loader.js
```

## 🚀 Производительность

```
┌─────────────────────────────────────────────┐
│  ОПТИМИЗАЦИИ                                │
└─────────────────────────────────────────────┘

1. CSS Transitions
   └─→ GPU ускорение вместо JS анимаций

2. Passive Event Listeners
   └─→ Не блокируют скролл

3. Debounce (300ms)
   └─→ Уменьшает количество вызовов поиска

4. will-change
   └─→ Подсказка браузеру для оптимизации

5. Lazy Loading
   └─→ nav-enhanced.js загружается после nav.html

6. CSS containment
   └─→ Изолирует стили для лучшей производительности
```

## 📱 Touch Events Flow

```
┌─────────────────────────────────────────────┐
│  ОБРАБОТКА TOUCH СОБЫТИЙ                    │
└─────────────────────────────────────────────┘

touchstart
├─→ Сохраняем начальную позицию (X, Y)
├─→ Проверяем зону активации (30px от края)
└─→ Устанавливаем флаг isSwiping = true

touchmove
├─→ Вычисляем deltaX, deltaY
├─→ Проверяем направление (горизонтальный?)
├─→ Применяем transform к меню
└─→ Добавляем класс .swiping

touchend
├─→ Проверяем расстояние (> 50px?)
├─→ Открываем/закрываем меню
├─→ Вибрация (haptic feedback)
├─→ Сбрасываем transform
└─→ Убираем класс .swiping
```

## 🔄 Event Flow

```
┌─────────────────────────────────────────────┐
│  СОБЫТИЯ И ОБРАБОТЧИКИ                      │
└─────────────────────────────────────────────┘

DOMContentLoaded
└─→ nav-loader.js
    └─→ loadNav()
        ├─→ fetch nav.html
        ├─→ replace {{BASE}}
        ├─→ insert HTML
        ├─→ highlightActive()
        ├─→ initMobileMenu()
        └─→ dispatch 'navLoaded'

navLoaded
└─→ nav-enhanced.js
    └─→ initAll()
        ├─→ initSwipeGestures()
        ├─→ initScrollProgress()
        ├─→ initSearch()
        ├─→ initUserProfile()
        ├─→ initAnimations()
        ├─→ highlightActivePage()
        └─→ initKeyboardShortcuts()

scroll
└─→ updateScrollProgress()
    ├─→ calculate percentage
    └─→ update progress bar width

keydown
├─→ Ctrl+K → focus search
└─→ ESC → close mobile menu

touchstart/touchmove/touchend
└─→ handleSwipeGestures()
    ├─→ detect swipe direction
    ├─→ update menu position
    └─→ open/close menu
```
