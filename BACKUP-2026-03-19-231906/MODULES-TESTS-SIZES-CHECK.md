## 📊 Найденные проблемы

### Страницы модулей (module-*.html)

- ❌ `.container` - `max-width: 900px` (должно быть 1000px)
- ❌ Footer `.container` - `max-width: 1200px` (должно быть 1000px)

### Страницы тестов (test-*.html)

#### test-1.html
- ❌ `.container` - `max-width: 900px` (должно быть 1000px)
- ❌ Footer `.container` - `max-width: 1200px` (должно быть 1000px)
- ⚠️ Сломанный padding в body: `$180px 20px` (должно быть `padding: 80px 20px`)

---

## 🔧 Что нужно исправить

### На всех страницах модулей и тестов:

1. **Основной контейнер**
   ```css
   .container {
     max-width: 1000px;  /* было: 900px */
   }
   ```

2. **Footer контейнер**
   ```css
   .footer .container {
     max-width: 1000px;  /* было: 1200px */
   }
   ```

3. **Body padding** (если сломан)
   ```css
   body {
     padding: 80px 20px 40px;  /* исправить если: $180px 20px */
   }
   ```
