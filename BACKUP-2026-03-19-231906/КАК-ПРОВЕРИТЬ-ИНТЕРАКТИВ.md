# 🎯 Как проверить интерактивные модули

## Откройте страницу

Скопируйте этот путь в адресную строку браузера:
```
file:///C:/DispatcherTraining/pages/doc-module-1.html
```

Или просто откройте файл двойным кликом:
```
C:\DispatcherTraining\pages\doc-module-1.html
```

## Что вы увидите

### 📚 Три блока с контентом:
1. **Роль диспетчера в грузоперевозках**
2. **Основные обязанности диспетчера**
3. **Необходимые навыки и качества**

### 💼 После каждого блока:
- **Case Study** (фиолетовый блок с иконкой 💼) - реальная ситуация из жизни диспетчера
- **Quick Check** (серый заблокированный блок с замком 🔒) - вопрос для проверки знаний

## Как это работает

### Шаг 1: Прочитайте контент
Прочитайте или прослушайте материал в блоке

### Шаг 2: Нажмите кнопку
Найдите кнопку **"🎧 I've finished listening"** и нажмите на неё

### Шаг 3: Квиз разблокируется
Серый блок Quick Check станет активным (исчезнет замок 🔒)

### Шаг 4: Ответьте на вопрос
Выберите один из трех вариантов ответа

### Шаг 5: Получите обратную связь
- ✅ **Правильный ответ** - зеленая рамка + объяснение
- ❌ **Неправильный ответ** - красная рамка + подсказка

## Примеры вопросов

### Квиз 1: Приоритизация задач
**Вопрос**: Какая задача должна быть выполнена в первую очередь?
**Правильный ответ**: Связаться с получателем о задержке груза

### Квиз 2: Расчет ставки
**Вопрос**: Какая минимальная ставка будет выгодной для груза Chicago → Dallas?
**Правильный ответ**: $2.40/mile ($2,880 total)

### Квиз 3: Deadhead Miles
**Вопрос**: Какая реальная ставка за милю с учетом пустого пробега?
**Правильный ответ**: $2.27/mile

## Мобильная версия

Откройте страницу на телефоне:
- Кнопки должны быть крупными (легко нажимать)
- Текст должен быть читаемым
- Всё должно помещаться на экран

## Если что-то не работает

### Проблема: Квиз не разблокируется
**Решение**: Проверьте консоль браузера (F12) на ошибки

### Проблема: Стили не применяются
**Решение**: Убедитесь, что файл `interactive-modules.css` находится в корне проекта

### Проблема: Ответы не сохраняются
**Решение**: Проверьте, что localStorage включен в браузере

### Проблема: Иконки не видны
**Решение**: Это эмодзи, они должны работать во всех современных браузерах
