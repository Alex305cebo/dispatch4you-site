🍞 Breadcrumbs

Хлебные крошки для навигации по сайту

📍 Что такое Breadcrumbs?

Breadcrumbs (хлебные крошки) — это навигационный элемент, который показывает путь от главной страницы до текущей. Помогает пользователям понять, где они находятся на сайте и быстро вернуться на предыдущие уровни.

Пример: Главная › Курсы › Модуль 1 › Введение

✨ Возможности

🎯 Автоматическое создание на основе URL
🎨 Красивый дизайн с иконками
📱 Полная адаптивность
✨ Плавные анимации
🔗 Кликабельные ссылки
🎭 Подсветка текущей страницы

🚀 Как использовать

Breadcrumbs загружаются автоматически через nav-loader.js. Никаких дополнительных действий не требуется!

Добавить свою страницу:

// В breadcrumbs.js или в своем коде
window.Breadcrumbs.addPageName(
    'my-page.html',      // Имя файла
    'Моя страница',      // Название
    '🎨'                 // Иконка (опционально)
);

Скрыть breadcrumbs на странице:

<body class="no-breadcrumbs">
    <!-- Breadcrumbs не будут показаны -->
</body>

Компактный режим:

// Добавить класс через JavaScript
document.querySelector('.breadcrumbs')
    .classList.add('compact');

📱 Адаптивность

  • Desktop (> 1024px): Полные названия с иконками
  • Tablet (768px - 1024px): Уменьшенные отступы
  • Mobile (480px - 768px): Компактный вид
  • Small Mobile (< 480px): Только иконки (текст скрыт, кроме текущей страницы)

🔧 Технические детали

  • Автоматическое определение пути на основе URL
  • Умное форматирование названий страниц
  • Поддержка вложенных папок
  • Анимация появления элементов по очереди
  • Glassmorphism эффект с blur
  • Фиксированное позиционирование под navbar
  • Accessibility (ARIA labels, focus states)

🎨 Цветовая схема

/* Основные цвета */
background: rgba(15, 23, 42, 0.8)  /* Темный фон */
border: rgba(255, 255, 255, 0.08)  /* Светлая рамка */

/* Ссылки */
color: #94a3b8                      /* Серый текст */
hover: #e0e7ff                      /* Светлый при hover */

/* Текущая страница */
background: linear-gradient(
    135deg,
    rgba(99, 102, 241, 0.2),
    rgba(139, 92, 246, 0.2)
)
color: #f1f5f9                      /* Белый текст */

⚙️ JavaScript API

// Создать breadcrumbs программно
const crumbs = window.Breadcrumbs.create();

// Отрендерить в HTML
const html = window.Breadcrumbs.render(crumbs);

// Вставить в DOM
window.Breadcrumbs.insert();

// Добавить новое название страницы
window.Breadcrumbs.addPageName(
    'file.html',
    'Название',
    '🎯'
);
