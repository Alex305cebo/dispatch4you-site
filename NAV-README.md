## 🔧 Что нужно настроить

### Включить профиль пользователя

```javascript
document.addEventListener('navLoaded', () => {
    const profile = document.getElementById('navUserProfile');
    profile.style.display = 'flex';
});
```

### Настроить поиск

Открой `nav-enhanced.js` и измени функцию `performSearch()`:

```javascript
function performSearch(query) {
    window.location.href = `/search?q=${encodeURIComponent(query)}`;
}
```

| `Свайп →` | Закрыть меню (mobile) |

## 📱 Свайп-жесты

### Открыть меню:
1. Проведи пальцем от **правого края** экрана **влево**
2. Или нажми кнопку ☰

### Закрыть меню:
1. Проведи пальцем **вправо** в открытом меню
2. Или нажми на затемненную область
3. Или нажми кнопку ✕

### Haptic feedback:
- Телефон вибрирует при открытии/закрытии

## 🐛 Решение проблем

### Свайп не работает
- Проверь, что открываешь на touch-устройстве
- Убедись, что `nav-enhanced.js` загружен

### Поиск не работает
- Настрой функцию `performSearch()` в `nav-enhanced.js`

### Профиль не виден
- Включи его через код выше
- По умолчанию скрыт
