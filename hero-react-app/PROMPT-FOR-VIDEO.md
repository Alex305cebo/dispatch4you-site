# Промт для генерации видео/изображения (Nano Banano)

## Для видео (рекомендуется):

```
Создай кинематографическое видео длительностью 10-15 секунд для hero-секции сайта обучения диспетчеров грузоперевозок.

Стиль: Современный, технологичный, профессиональный, темная цветовая гамма

Визуальные элементы:
- Анимированная карта США с подсвеченными маршрутами грузоперевозок
- Светящиеся линии дорог, соединяющие города (синий, фиолетовый, голубой цвета)
- Движущиеся точки по маршрутам, символизирующие грузовики
- Пульсирующие маркеры городов (Нью-Йорк, Лос-Анджелес, Чикаго, Даллас, Атланта)
- Голографические элементы интерфейса Load Board (полупрозрачные панели с данными)
- Цифровые индикаторы: ставки перевозок, расстояния, время в пути
- Частицы и световые эффекты, создающие ощущение высоких технологий

Цветовая палитра:
- Основной фон: темно-синий градиент (#0a0e1a → #1a1f35 → #2a1f3f)
- Акценты: электрический синий (#6366f1), фиолетовый (#8b5cf6), голубой (#22d3ee)
- Свечение: мягкое неоновое свечение вокруг линий и точек

Атмосфера: Футуристичная, динамичная, высокотехнологичная логистика

Движение камеры: Медленный плавный зум от общего плана карты к деталям маршрутов

Формат: MP4, 1920x1080, 30fps, зацикленное видео (loop)
```
